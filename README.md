# 📝 Todo List App

A simple and clean Todo List web application to manage your daily tasks easily.

---

## 🚀 Live Demo

🔗 https://mahdimix1386-design.github.io/Todo-List/

---

## 📌 Features

* ➕ Add new tasks
* ❌ Delete tasks
* ✅ Mark tasks as completed
* 💾 Save tasks (if using localStorage)
* 🎨 Simple and user-friendly UI

---

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📥 Installation

To run this project locally:

```bash
git clone https://github.com/mahdimix1386-design/Todo-List.git
cd Todo-List
```

Then open `index.html` in your browser.

---

## ▶️ Usage

* Type your task in the input field
* Click "Add" to save it
* Click on a task to mark it as completed
* Use delete button to remove tasks
